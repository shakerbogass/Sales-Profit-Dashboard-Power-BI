
# Sales & Profit Dashboard | Power BI

This project is an interactive Power BI dashboard designed to analyze sales and profit performance from multiple perspectives.

## Key Features
- Total Sales, Profit, Quantity, Discount, and COGS
- Profit analysis by Region, Person, and Ship Mode
- Geographic analysis by country
- Interactive slicers for filtering

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Modeling
